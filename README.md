### Quickshell
Network Widget requires the NetworkMonitorPlugin which was developed by yours truly.   
Find it here: https://github.com/vnsnippets/qt-plugin-network-monitor

You will need to set the `QML_IMPORT_PATH` environment variable to point to where you drop the plugin (after build). Personally, I set it through my `hyprland.conf` (in this repository itself).
